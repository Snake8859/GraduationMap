# 毕业地图

个人兴趣小应用，最初设想做一个小程序，能记录学长学姐们的毕业去向，类似于每年毕业季的毕业流向图。

前端：小程序+weui+高德地图API

后端：servlet+hibernate+spring

本应用，有一段时间暂未维护和更新，该版本出来不少的bug和问题，待后期进行维护和更新时解决。
